# Scrum Product Backlog

## User Stories

### User Story 1
Como jogador, quero desligar o som do jogo nas definições para jogar sem áudio.

#### Passos testados
1. Aceder ao website do Battleship
2. Aceitar cookies
3. Abrir definições
4. Desligar o som
5. Verificar que o estado do botão mudou

---
